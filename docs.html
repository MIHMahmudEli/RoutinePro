Meet RoutinePro

The intelligent university schedule builder — designed for students who want zero conflicts, maximum control, and a routine that actually works for them.

Instant Generation Multiple Scenarios Themed UI Focus Mode Ramadan Mode PNG Export

The Problem

Manually checking section conflicts is tedious and error-prone. One mistake means sitting through a broken schedule all semester.

How We Solve It

Our generation engine checks every possible section combination and only shows you conflict-free results—done in milliseconds.

The Result

A perfect, shareable routine in under 5 minutes — with a beautiful UI that makes course planning feel premium.

📋 Course Queue

Building Your Queue

Adding Courses

Click any suggestion in the search dropdown. The course is added to the top of your queue and appears in the sidebar on the left.

Removing Courses

Hover over any course card in the sidebar queue and click the ✕ remove button in the top-right of the card.

Pinning a Section

Already decided on a specific section? Click the 🔒 lock button to "pin" it. The generator will always use this exact section regardless of other scenarios.

Manual Entry Mode

Have a class that isn't in the database (e.g., a lab, thesis, or off-campus session)? Use Manual Entry to add custom time blocks with any name, days, and time range.

How Sorting Works

RoutinePro sorts by total weekly waiting gap — the total free time between your classes. Scenario #1 always has the tightest, most time-efficient schedule. The further you go, the more scattered the timetable.

🌙 Ramadan Mode

Ramadan Mode

University schedules shift during Ramadan. Instead of manually adjusting each class, the app transforms your entire routine automatically.

Regular Schedule

Class 1 08:00 - 09:30
Class 2 09:40 - 11:10
Class 3 11:20 - 12:50

Ramadan Schedule

Class 1 08:00 - 09:00
Class 2 09:10 - 10:10
Class 3 10:20 - 11:20

This feature is admin-controlled. The toggle is only visible when an administrator has enabled it for the current period. If you don't see the 🌙 button, Ramadan Mode is not currently needed.
